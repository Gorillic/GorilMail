# GorilMail

GorilMail is a standalone World of Warcraft Retail mailbox addon by **Goril Labs**.

## Features
- Fast inbox list with sender, subject, money, COD, and attachment indicators
- `Collect All` with safe queue-based progression and validation
- Single-row collect action for quick manual control
- Send panel with recipient, subject, body, attachments, COD/Gold
- `Fill Similar` to quickly populate matching send attachments
- Separate Destroy panel with candidate scan (Disenchant / Milling)
- `Destroy Next`, `Skip`, refresh, and selectable rows in Destroy flow
- Two built-in UI themes: Horde and Alliance
- Session-level collect summary in UI/chat

## Commands
- `/gorilmail destroy` -> toggle Destroy panel
- `/gmail destroy` -> toggle Destroy panel

## Requirements
- World of Warcraft Retail

## Installation
1. Copy `GorilMail` folder into `_retail_/Interface/AddOns/`
2. Restart game or run `/reload`
3. Enable `GorilMail` in AddOns list
